# Data Order Limitation

For each data generation order, the following limitations are enforced:

| Configuration Name        | Value     |
|---------------------------|-----------|
| Entity Generation Timeout | 30 Minute |
| Max Index Collision       | 400       |
| Data Generate Timeout     | 60 Minute |
| Retry                     | 2         |
